# Sistema de Registro Nacional de Pessoas

## Objetivo do Projeto

O Sistema de Registro Nacional de Pessoas é uma aplicação desenvolvida em Python com o objetivo de centralizar e gerenciar o registro de pessoas em um país, atendendo às necessidades da administração pública relacionadas à identificação civil.

O sistema permite o gerenciamento de três tipos de trâmites:

* Emissão de documentos de identificação para cidadãos;
* Concessão de autorizações de residência para estrangeiros;
* Concessão de autorizações de refúgio.

Cada trâmite possui informações comuns, como dados pessoais, fotografia, impressão digital e data de emissão, além de atributos específicos, como país de origem, motivo do refúgio ou data de vencimento da autorização de residência.

Além disso, o projeto implementa diferentes perfis de usuários (Administrador, Supervisor, Funcionário de Registro e Solicitante), permitindo operações de cadastro, consulta, atualização, renovação e revogação de documentos, bem como a geração de relatórios básicos e o controle de acesso baseado em papéis.

---

## Funcionalidades

* Cadastro de documentos de identificação.
* Cadastro de autorizações de residência.
* Cadastro de autorizações de refúgio.
* Busca por número do documento.
* Busca por nome.
* Busca por tipo de processo.
* Renovação de documentos.
* Revogação de documentos.
* Geração de relatórios básicos.
* Controle de acesso por perfis.
